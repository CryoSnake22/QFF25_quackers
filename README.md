# QFF25_quackers

## Problem Definition
Optimize circuit embedding of Quantum Fourrier Transform.

## Approach
1. Implement QFT circuit
2. Use simulation to get ideal result
3. Run default transpiler and compare result w noise to ideal
4. Implement a custom transpiler and compare

## How to Run

Installation

```
git clone git@github.com:CryoSnake22/QFF25_quackers.git
```

Then create a virtual environment and install dependencies:

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Run our notebook!
